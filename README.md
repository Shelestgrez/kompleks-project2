# Комплекс-проект — ИС управления проектами

Веб-приложение: учёт проектов (архитектура, инженерные изыскания, испытания), задач, договоров, документов и записей испытаний.

## Возможности интерфейса

- Тёмная и светлая тема (сохраняется в браузере).
- Панель: диаграммы по статусам задач и проектов, счётчики документов и испытаний, лента недавних задач.
- Проекты: поиск, фильтр по статусу, избранное (локально), экспорт списка в CSV, горячая клавиша **`/`** для фокуса поиска.
- Карточка проекта: копирование кода в буфер, задачи в виде **таблицы** или **канбана** с перетаскиванием между колонками.

## Запуск

### Быстрый запуск на Windows (рекомендуется)

- Для разработки: запустите `Комплекс-проект-Разработка.bat`
- Для установки + сборки + запуска preview: `Комплекс-проект-Запуск.bat`
- Альтернатива через PowerShell: `.\run-dev.ps1`

Скрипты автоматически:
- проверяют наличие `Python 3.11+`, `Node.js 20+`, `npm`;
- создают `backend/.venv` (если его ещё нет);
- ставят backend/frontend зависимости;
- запускают API (`127.0.0.1:8000`) и frontend (`127.0.0.1:5173`).

**1. Backend** (Python 3.11+, порт **8000**):

```bash
cd backend
python -m pip install -r requirements.txt
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

При первом запуске создаётся SQLite `backend/data.db` и демо-данные.

**2. Frontend** (Node 20+, порт **5173**):

```bash
cd frontend
npm install
npm run dev
```

Интерфейс проксирует `/api` и `/files` на `http://127.0.0.1:8000`. Откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

## Сборка фронтенда

```bash
cd frontend
npm run build
```

Статика — в `frontend/dist` (для продакшена настройте reverse-proxy на API).

## Стек

- Backend: FastAPI, SQLAlchemy, SQLite, JWT (bcrypt + jose).
- Frontend: React 19, TypeScript, Vite, React Router, Tailwind CSS v4, @hello-pangea/dnd (канбан).
